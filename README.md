# US Travel Map

A React + Vite website where you can:

- Click US states on a map to mark as visited (green)
- Check/uncheck states from a list on the right
- Auto-save your selections in localStorage (persists after reload)

## Requirements

- Node.js 18+ (recommended)

## Setup

```bash
npm install
npm run dev
```

Then open the local URL shown by Vite.
